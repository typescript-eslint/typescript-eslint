This rule extends the base [`eslint/consistent-return`](https://eslint.org/docs/rules/consistent-return) rule.
This version adds support for functions that return `void` or `Promise<void>`.

```ts
  if (flag) return;
  return foo();
}
```

```ts
  if (flag) return 42;
  return;
}
```
