Thank you for your interest in contributing to typescript-eslint! 💜

Although typescript-eslint's packages are already downloaded millions of times per month and power high profile projects across our industry, this is a 100% community-driven project.
From the second you install one of the packages from this monorepo, you are a part of that community.
So, from the bottom of our hearts, _thank you_ 💕.

## Getting Started

Please start by reading our [Code of Conduct](https://github.com/typescript-eslint/typescript-eslint/blob/main/CODE_OF_CONDUCT.md).
If you're using AI agents to write code, be sure to also read our [AI Contribution Policy](https://typescript-eslint.io/contributing/ai-policy).
And if you're new to open source, the [How to Contribute to Open Source](https://opensource.guide/how-to-contribute) guide is a helpful place to begin.
